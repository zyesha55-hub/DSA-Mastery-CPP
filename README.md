🚀 Projects Included
1. Stack Implementation (Array-based)
File: stack_implementation.cpp

Description: A complete implementation of the Stack data structure using Arrays and Object-Oriented Programming (OOP) principles.

Key Operations:

push(): Adds an element to the top of the stack.

pop(): Removes the top element.

peek(): Displays the topmost element without removing it.

isEmpty(): Checks if the stack is empty.

Logic: Follows the LIFO (Last In, First Out) principle.
